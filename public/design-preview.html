📌 MARK ⧉ COPY

#番号タップ（v2.2 はカード全体タップ）後の詳細＝1 件をじっくり読む場。見出し帯（.band）は #NNN・感情カラー声紋ヒーロー・スタンプをまとめ、その RECORD の感情スコア由来のグラデを背後に敷く（未解析なら無色）。STATS は LENGTH / DAY / CHARS（CHARS は文字数のカウントアップ）。本文は「。」で段落分割・行間を空け・一回り大きく濃く（--text-lg）、開いた瞬間に上から順にリビール。その RECORD の内容語（WORDS）を本文中で自動ハイライト（--yuzu-yellow ＋細い下線のみ。文言は変えない）。MARK / COPY はここに集約（COPY は Notion 移行期間限定の一時機能）。--ink 背景・白文字・罫線＋背景グラデのみで階層化（box-shadow なし）。

NG / OK

✗ NG ワード
  • 癒し・寄り添う・頑張ろう（muute っぽい）
  • 育つ・林・種・香り・果実（旧世界観の残骸）
  • やさしく・ふんわり・あなたらしく
  • 入力・テキスト・記録する（ツールっぽい）
  • 気づき・自分を知ろう（意識高い系）
✓ OK 方向
  • 話せ・出せ・黙るな（命令形）
  • 本物・ナマ・加工なし
  • RAW・RECORD・MARK・INDEX
  • 事実だけを置く（日本語）
  • 状態を断定する（英語）

用語定義NG 言い換え
LOGユーザー自身が出した記録の集約面（STATS / RECORDS）。デフォルトタブ。「ME」「マイページ」「INDEX」（旧称）
INSIGHTAI と集計が声を解釈して返す面（EMOTION / SIGNAL / WORDS / PATTERN / REPORTS）。「READ」「REPORT」（旧称）
RECORD FAB録音アクション。タブバー右の正円ボタン。タブ state を持たない。旧「TALK タブ」は廃止
#NNNユーザーの identity 兼通し番号。名前・アイコンの代替。ゼロ埋め3桁。「名前は無い。お前は #020 だ」
RECORD / RECORDS1 件の声の記録 / その一覧。「投稿」「日記」「テキスト」
MARK / MARKED（フィルタ）RECORD に刻印を打つトグル / MARK 済みのみのフィルタ。「お気に入り」「いいね」「ブックマーク」「PINNED」
COPY本文 + #NNN + 日時をクリップボードへ。Notion 移行期間限定の一時機能。⚠️ 将来削除。恒久 UI ではない
STATSLOG 上部の 3 スタッツ RECORDS / MINUTES / STREAK。DAY は INDEX 詳細モーダル側。「DAYS. NO SKIP.」は使わない
MINUTES総録音分数。声の積み上げ量。旧「SINCE（登録日数）」を置換
STREAK連続投稿日数。—
SILENCE録音がなかった日に刻まれる印（SILENCE）。沈黙も記録の一部
