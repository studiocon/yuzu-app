# PEACH — PRD

> 声は、種。つぶやきは、実る。

PEACH は、声でつぶやくと思考が育っていくサービスです。キーボードも整理も不要。長押しして話すだけで投稿される。あなたのつぶやきは、いつか他の誰かのつぶやきと勝手に繋がり、予想もしなかったアイデアになって戻ってくる。人をフォローするんじゃない。**考えが出会う場所**です。

---

## 1. プロダクト概要

### コンセプト

- **声は種** — 長押しして話すだけで投稿
- **育つ** — マイページで思考が時系列に蓄積
- **繋がる** — 関連ポストがAIで自動リンク（V3）
- **実る** — AIが要約・音声で読み上げ（V2以降）

メモ帳は自分のため。SNSは他人のため。**PEACH は思考のため**。

### 他サービスとの比較

| 比較対象 | PEACH との違い |

| メモ帳 / Notion | 書く前提・整理前提。PEACH は声でこぼすだけ |
| Claude / ChatGPT | 会話が消える。PEACH は蓄積・繋がる |
| Twitter / X | バズ圧力・人フォロー。PEACH は考えが繋がる |
| 日記アプリ | 続かない。PEACH は長押し1アクション |

## 2. 機能要件（MVP v1）

- **3タブ構成**：タイムライン / はなす（デフォルト）/ マイページ。下部にiOS 26 Liquid Glass 風のフローティングタブバー。
- **マイクボタン長押し録音**：押した瞬間に録音開始、離すと終了。
- 指を離すと自動で文字起こし（STT）→ 即投稿 → タイムラインに自動遷移。
- はなす画面は粒子アニメーションで "聴いている" 気配を演出（待機/録音/変換でドットの挙動が変わる）。
- **STT**：ElevenLabs Scribe v2 で音声→テキスト変換。
- **投稿保存**：Vercel KV（Upstash Redis）で全ユーザー間共有。
- **タイムライン**：全投稿を新着順に表示（歪んだ楕円カード）。
- **果物スタンプ**：🍑 わかる / 🍋 ちょっとすっぱい / 🌱 育ちそう / 🫐 じんわりくる / 🍒 好き。
- **ゲストモード**：投稿・閲覧・リアクション可能（cookie で sessionId を保持）。
- マイページは現状プレースホルダー。

## 6. 制約・設計思想

意図的に削ぎ落としています。

- 音声入力のみ（テキスト入力欄なし）
- 編集不可
- 削除不可
- フォロー / いいね / リポストなし

「整理する前」の生っぽい思考に価値があるという思想に基づいています。

## 7. Eval（品質基準・テストケース）

> PRDの要件変更時は、対応するEvalケースも同時に更新すること。

### STT（音声→テキスト変換）

**成功基準：** 3秒以内にテキストが返る。日本語として読める文章になっている。

**テストケース：**
- 30秒の日本語音声 → 句読点ありのテキスト、3秒以内に返る
- 1秒未満の短い音声 → エラーにならずに処理される
- 無音の音声 → 空文字またはエラーメッセージ（クラッシュしない）

### 投稿保存（Upstash Redis）

**成功基準：** 投稿後1秒以内にタイムラインに反映される。

**テストケース：**
- 投稿 → タイムラインの先頭に表示される
- 100文字の投稿 → そのまま保存・表示される
- 連続投稿（5秒以内に2回）→ 両方保存される

### 果物スタンプ

**成功基準：** タップ後即座に反映。カウントが正確。

**テストケース：**
- 🍑をタップ → カウントが1増える
- 同じスタンプを2回タップ → カウントが1に戻る（トグル）
- 複数スタンプをタップ → それぞれ独立してカウントされる

### AIインサイト生成（V2）

**成功基準：** 自然な日本語・200文字以内・ポジティブな表現で返る。

**テストケース：**
- ポジティブな投稿10件 → 前向きな要約が返る
- ネガティブな投稿10件 → 断定的なネガティブ表現にならない
- 雑多なトピックの投稿 → テーマを整理して返す
- NGケース: 「あなたはネガティブです」のような断定 → 不合格

### TTS（テキスト→音声読み上げ）（V2）

**成功基準：** 5秒以内に音声が再生される。自然な日本語イントネーション。

**テストケース：**
- 100文字のインサイト → 5秒以内に再生開始
- 記号・絵文字を含むテキスト → クラッシュしない
- NGケース: 音声が途切れる・文字化けした読み上げ → 不合格
